# @bch-stealth/pool-hash-fold

CashAssembly/CashVM covenant programs and hash-fold state transition logic (including v1.1 unlocking stack that includes `noteHash32` and `proofBlob32`).

## What lives here

- Source: `packages/pool-hash-fold/src`
- Build output (if applicable): `packages/pool-hash-fold/dist`
- Tests (if applicable): `packages/pool-hash-fold/tests`

## How to build

From repo root:

```bash
yarn workspace @bch-stealth/pool-hash-fold build
```
