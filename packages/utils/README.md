# @bch-stealth/utils

Shared utilities (bytes/hex helpers, cashaddr helpers, logging, small parsers).

## What lives here

- Source: `packages/utils/src`
- Build output (if applicable): `packages/utils/dist`
- Tests (if applicable): `packages/utils/tests`

## How to build

From repo root:

```bash
yarn workspace @bch-stealth/utils build
```

## Notes

- This package is part of a Yarn workspaces monorepo.
- See the root `README.md` for end-to-end setup and the Phase 2 demo flow.
