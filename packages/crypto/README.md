# @bch-stealth/crypto

Cryptographic primitives and helpers used across packages (hashing, key ops, etc.).

## What lives here

- Source: `packages/crypto/src`
- Build output (if applicable): `packages/crypto/dist`
- Tests (if applicable): `packages/crypto/tests`

## How to build

From repo root:

```bash
yarn workspace @bch-stealth/crypto build
```
