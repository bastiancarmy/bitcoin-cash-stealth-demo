# @bch-stealth/gui

Electron + React GUI wrapping `bchctl` operations (profiles, send, scan, pool init/import/withdraw).

## What lives here

- Source: `packages/gui/src`
- Build output (if applicable): `packages/gui/dist`
- Tests (if applicable): `packages/gui/tests`

## How to build

From repo root:

```bash
yarn workspace @bch-stealth/gui build
```

## Notes

- This package is part of a Yarn workspaces monorepo.
- See the root `README.md` for end-to-end setup and the Phase 2 demo flow.
