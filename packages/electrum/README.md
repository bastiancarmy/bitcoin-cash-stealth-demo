# @bch-stealth/electrum

Electrum/Fulcrum client layer used for chain queries and transaction fetching/broadcast.

## What lives here

- Source: `packages/electrum/src`
- Build output (if applicable): `packages/electrum/dist`
- Tests (if applicable): `packages/electrum/tests`

## How to build

From repo root:

```bash
yarn workspace @bch-stealth/electrum build
```

## Notes

- This package is part of a Yarn workspaces monorepo.
- See the root `README.md` for end-to-end setup and the Phase 2 demo flow.
