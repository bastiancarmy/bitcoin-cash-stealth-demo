# @bch-stealth/pool-shards

Pool shard transaction builder and covenant interaction logic (deposit/import/withdraw) plus parsing utilities.

## What lives here

- Source: `packages/pool-shards/src`
- Build output (if applicable): `packages/pool-shards/dist`
- Tests (if applicable): `packages/pool-shards/tests`

## How to build

From repo root:

```bash
yarn workspace @bch-stealth/pool-shards build
```

## Notes

- This package is part of a Yarn workspaces monorepo.
