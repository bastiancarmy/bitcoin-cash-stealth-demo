# @bch-stealth/pool-state

Type definitions and persistence helpers for profile state (`state.json`), including stealthUtxos, deposits, shards, and withdrawals.

## What lives here

- Source: `packages/pool-state/src`
- Build output (if applicable): `packages/pool-state/dist`
- Tests (if applicable): `packages/pool-state/tests`

## How to build

From repo root:

```bash
yarn workspace @bch-stealth/pool-state build
```

## Notes

- This package is part of a Yarn workspaces monorepo.
- See the root `README.md` for end-to-end setup and the Phase 2 demo flow.
