# @bch-stealth/rpa-scan

Scanning logic for discovering inbound stealth P2PKH outputs and writing them into state.

## What lives here

- Source: `packages/rpa-scan/src`
- Build output (if applicable): `packages/rpa-scan/dist`
- Tests (if applicable): `packages/rpa-scan/tests`

## How to build

From repo root:

```bash
yarn workspace @bch-stealth/rpa-scan build
```

- This package is part of a Yarn workspaces monorepo.
