# @bch-stealth/cli

Command-line interface (`bchctl`) for wallet ops, RPA send/scan, pool stage/import/withdraw, and debugging utilities.

## What lives here

- Source: `packages/cli/src`
- Build output (if applicable): `packages/cli/dist`
- Tests (if applicable): `packages/cli/tests`

## How to build

From repo root:

```bash
yarn workspace @bch-stealth/cli build
```

## Notes

- This package is part of a Yarn workspaces monorepo.
- See the root `README.md` for end-to-end setup and the Phase 2 demo flow.
