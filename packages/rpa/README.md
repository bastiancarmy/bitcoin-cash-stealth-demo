# @bch-stealth/rpa

Reusable Payment Address (RPA)/paycode derivation logic and related constants.

## What lives here

- Source: `packages/rpa/src`
- Build output (if applicable): `packages/rpa/dist`
- Tests (if applicable): `packages/rpa/tests`

## How to build

From repo root:

```bash
yarn workspace @bch-stealth/rpa build
```

## Notes

- This package is part of a Yarn workspaces monorepo.
