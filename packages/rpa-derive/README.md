# @bch-stealth/rpa-derive

Derivation utilities used for generating scan/spend keys and one-time keys from RPA context.

## What lives here

- Source: `packages/rpa-derive/src`
- Build output (if applicable): `packages/rpa-derive/dist`
- Tests (if applicable): `packages/rpa-derive/tests`

## How to build

From repo root:

```bash
yarn workspace @bch-stealth/rpa-derive build
```

## Notes

- This package is part of a Yarn workspaces monorepo.
- See the root `README.md` for end-to-end setup and the Phase 2 demo flow.
